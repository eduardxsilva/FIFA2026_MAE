# FIFA 2026 Analytics — Liquid Glass Edition

Interface Streamlit com visual liquid glass, navegação moderna e ícones outline.

## Arquivos principais

- `streamlit_app.py`
- `fifa2026_core.py`
- `requirements.txt`
- `runtime.txt`
- `.streamlit/config.toml`

## Rodar localmente

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Deploy no Streamlit Cloud

Suba todos os arquivos na raiz do GitHub e use:

```text
Main file path: streamlit_app.py
```

O projeto usa `streamlit-option-menu` para navegação com ícones outline e SVGs inline para os cards.
